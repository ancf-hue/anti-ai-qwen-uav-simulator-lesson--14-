🚁 Механик БПЛА — Интерактивный 3D-симулятор ремонта дронов



📖 О проекте
"Механик БПЛА" — это обучающий 3D-симулятор, который погружает вас в роль инженера по ремонту беспилотных летательных аппаратов. Проект создан в рамках Анти-ИИ Академии (Урок 14) и сочетает в себе увлекательный игровой процесс с элементами профессиональной диагностики.

Получите повреждённый дрон, осмотрите его со всех сторон, найдите неисправности, подберите правильные инструменты и выполните качественный ремонт. После устранения всех поломок — проведите тестовый полёт!

🎯 Основные возможности
🖱️ Интерактивное 3D-управление: Вращайте дрон мышью, приближайте и отдаляйте камеру для детального осмотра

🔧 Система профессиональных инструментов: Отвёртка, паяльник, ключ, мультиметр и плоскогубцы — каждый для своего типа работ

📋 Три уровня сложности:

🟢 Стажёр — 3 поломки

🟡 Техник — 5 поломок

🔴 Мастер — 7 поломок

📊 Реальная статистика: Счётчик починенных деталей, таймер миссии, показатель точности

🏆 Система достижений: Модальное окно победы с детальной статистикой

🎵 Атмосферное сопровождение: Звуковые эффекты для каждого действия и фоновое музыкальное оформление

🛠️ Технологии
Three.js (r128) — 3D-визуализация и рендеринг

Чистый CSS — стеклянные эффекты (glassmorphism), адаптивная верстка

Нативный JavaScript — без внешних зависимостей, вся логика в одном файле

Web Audio API — создание звуковых эффектов на лету
